# Fire Dashboard

A lightweight personal information dashboard designed for a 2019 Amazon Fire 7 running Fully Kiosk Browser.

The current version includes:

- Live clock and date
- Norwegian Gem countdown
- A device-local "Today's focus" field
- A placeholder for live Ballyferriter weather

The focus text is stored in the browser's local storage and is not sent to GitHub.

## Editing

Edit `index.html`, commit the change, and GitHub Pages will publish the updated dashboard at the same URL.

Do not add private calendar-feed URLs, Home Assistant tokens, passwords, or API secrets to this public repository.
